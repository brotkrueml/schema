Version 2 of the extension supports TYPO3 v10 LTS and TYPO3 v11 LTS and needs at least PHP 7.4.

Latest version 1.x supports TYPO3 v9 LTS, TYPO3 v10 LTS, TYPO3 v11 and PHP 7.2+.
